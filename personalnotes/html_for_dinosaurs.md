
#### Intro

An HTML document from the 1990s would look similar to one from today. The emphasis however has changed from having a focus on content (static websites) to having a focus on dynamic web applications (focus on interaction).

With basic HTML you can get the workings but it's not fully semantic, meaning that the tags don't communicate the meaning of the content as well as they could. HTML5 (introduced in 2008) provides new elements to improve document semantics. 
    - This is useful as web browsers, search engines and screen readers all depend on semantic HTML to function properly
    - This also allows for keyboard shortcuts to work properly by providing a relationship between components
    

WAI-ARIA (Web Accessibility Initiative -Accessible Rich Internet Applications) is a set of attributes that makes HTML more accessible when semantic tabs aren't enough.
